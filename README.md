# 5590 AGI Final Project

This repository contains the code and final report for our 5590 AGI final project.

```text
.
├── 5590-AGI-code/
│   ├── Model/
│   ├── Trainer/

│   ├── main_k_fold_CDR.bash
│   └── parameters.py
├── 5590_AGI_report.pdf
└── README.md
```

## How to Run

Go to the code directory:

```bash
cd 5590-AGI-code
```

Run the main experiment script:

```bash
bash main_k_fold_CDR.bash
```

This script runs the k-fold CDR experiment using the provided project code.
